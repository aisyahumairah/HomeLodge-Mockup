Permissions

System-wide permission matrix

Permission Admin Guest Description
booking:view View own bookings
booking:create Create new bookings
booking:edit Edit any booking (admin only)
booking:delete Delete bookings (admin only)
booking:cancel Cancel own bookings
payment:view View payment records
payment:manage Manage payments and refunds
user:view View all users
user:manage Create, edit, delete users
chat:use Send and receive chat messages
settings:manage Manage system settings
audit:view View audit logs
calendar:block Block dates on calendar
